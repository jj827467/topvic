私隱政策

拓域科技有限公司個人資料保護條例 | 最近更新: 2026年3月

本私隱政策說明了「拓域科技」在提供系統開發、AI 服務整合及海外 AI 代購服務時，如何收集、使用、存儲和保護您的個人資料，以及您對這些資料的權利。請仔細閱讀以了解我們對您私隱的承諾。

第一條（收集目的）

為順利提供及改進資訊科技與 AI 服務，本公司將依據澳門特別行政區第8/2005號法律《個人資料保護法》的規定，在適用的合法基礎上（如履行合同、當事人明確同意或正當利益等），收集、處理和使用您的個人資料。主要目的包括：

  1. 提供、預約及確認系統開發、AI 影片製作與服務代購。
  2. 評估專案需求、計算服務費用並處理付款。
  3. 核實客戶身份，開通並交付代購之軟體或 API 帳號。
  4. 進行專案溝通、技術支援與售後維護服務。
  5. 監控服務品質，處理客戶查詢、技術障礙與爭議。
  6. 履行法定義務（例如：配合司法或行政機關的要求、稅務審計）。
  7. 在取得您明確同意的情況下，用於市場行銷或推廣活動（如發送新技術資訊或優惠）。

第二條（收集資料範圍）

本公司在提供服務期間可能收集的個人與企業資料類型包括：

  1. 基本識別資訊： 您的姓名、聯絡電話、電子郵件地址、公司名稱及職位。
  2. 專案相關資訊： 系統需求文檔、測試數據、以及在開發或串接 AI API 過程中必要的業務邏輯資訊。
  3. 技術憑證資訊： 伺服器登入資訊（僅限授權範圍）、第三方 API 金鑰（如代購或系統串接所需）。
  4. 支付資訊： 支付方式及轉帳記錄（如信用卡付款，敏感卡號資訊將交由第三方支付網關處理，我們不作完整儲存）。
  5. 設備與技術資訊： 當您瀏覽我們的網站或使用我們開發的系統原型時，我們可能會收集 IP 位址、瀏覽記錄等以優化體驗。

※ 敏感資料：原則上，我們不會刻意收集《個人資料保護法》所定義的「敏感資料」（如政治信仰、宗教信仰、健康狀況等）。請您在提供測試數據時，亦避免包含真實的敏感個人資料。

第三條（資料共享與轉移）

原則：我們對您的商業機密與個人資料嚴格保密，絕不會隨意出售、交易或出租給無關的第三方。

必要的共享：僅在以下情況下，我們可能會在必要的範圍內共享您的資訊：

  1. 開發與技術團隊： 為完成專案，將必要資訊提供給參與該專案的內部工程師或簽署保密協議的協作夥伴。
  2. 服務供應商： 委託第三方供應商提供基礎設施支援，這些供應商將根據合約指示處理資料，並須承擔保密責任。
  3. 法律要求： 為遵守法律法規或回應政府機構的合法要求。

跨境傳輸：由於現代 IT 與 AI 服務的特性，您的資料可能需要傳輸至澳門境外（例如使用海外雲端伺服器或調用海外 AI API）。我們將確保該傳輸符合《個人資料保護法》的規定，並確保接收方具備適當的資料保護水平。

第四條（資料當事人的權利）

根據《個人資料保護法》，您有權就您的個人資料行使以下權利：

  1. 查閱權： 查詢我們是否持有您的個人資料及要求閱覽。
  2. 更正權： 要求更正或更新不準確或不完整的個人資料。
  3. 刪除權： 在特定條件下（如專案已結案且無維護合約），要求刪除您的聯絡資料或專案源碼。但依法需保留的財務或稅務記錄除外。

第五條（資料安全與保存期限）

安全措施：作為一間資訊科技公司，我們採用高標準的技術防護措施（包括 SSL 傳輸加密、資料庫加密、嚴格的存取權限控制及原始碼版控安全）來保護您的專案資料與個人資訊，防止未經授權的存取或外洩。

保存期限：我們僅在實現本政策所述目的所必需的期限內保留您的資料。專案交付並結束維護期後，除非法律另有規定，我們將根據客戶要求封存或安全銷毀相關業務數據。
